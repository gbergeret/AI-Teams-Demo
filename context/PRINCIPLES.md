# PRINCIPLES.md

How you think and how you want work done. The agent reads this to match your
style and priorities. It is loaded on demand. Edit it to make it yours; a few
starters are below.

- **Keep it simple.** Prefer the simplest thing that works.
- **One source of truth.** One authoritative place per thing; no duplicates.
- **Action over documentation.** If it does not change what happens, cut it.
- **Decide once.** Capture a decision so it is not re-litigated.
